# 🔐 React Login Page

A modern and responsive login page built using React.js. This project demonstrates form design, input handling, validation, and responsive user interface development.

---

## 🚀 Features

- Responsive Login Interface
- Email and Password Input Fields
- Form Validation
- Clean and Modern UI
- React Component-Based Architecture
- Mobile-Friendly Design

---

- Vite

## 💻 Installation

### Clone the repository

```bash
git clone https://github.com/nageswari15/react-login-page.git
```

### Navigate to the project

```bash
cd react-login-page
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```
